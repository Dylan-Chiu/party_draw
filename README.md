# party_draw

### 一、必做步骤
1、替换"resources/背景.png"（需为1920 * 1080）  
2、根据背景图调整"resources/main.css"文件内字体颜色  
3、调整"HelloController"下"list1_path"与"list2_path"（填写绝对路径，文本文件为gbk编码）  
4、电脑分辨率调整1920*1080 100%缩放

### 二、使用方法
#### 左下三个按钮：  
按住第一个：抽list1的名单  
按住第二个：抽list2的名单  
点按第三个：重新读取名单并重置
